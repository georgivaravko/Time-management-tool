# Time-management-tool
Ajanhallinta-sovellus TKT20019 kurssia varten Helsingin yliopistolla.

Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan menojaan.
Käyttäjä pystyy lisäämään prioriteetin menolleen.
Käyttäjä näkee sovellukseen hänelle tarkoitetut henkilökohtaiset menot, kuten uni ja lepo.
Käyttäjä pystyy etsimään menojaan hakusanalla.
Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja työtunneista per viikko ja käyttäjän lisäämät menot.
Käyttäjä pystyy valitsemaan menolleen yhden tai useamman luokittelun (esim. menon luokka, tuntimäärä per viikko, tärkeysjärjestys).
Käyttäjä pystyy hallitsemaan menojaan.
